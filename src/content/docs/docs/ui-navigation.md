---
title: UI Navigation
description: How to navigate the application interface
---

The application uses a responsive layout with a top navigation bar, an optional sidebar, and a language switcher.

## Top Navigation

The header at the top of every page contains:

- **Logo and title** — Click to return to the home page
- **Navigation sections** — Four dropdown menus organized by category
- **Language switcher** — Globe icon to change the UI language
- **Sidebar toggle** — Hamburger button (visible on smaller screens)
- **Current user** — Your account info (when connected)

### Navigation Sections

The navigation is divided into four sections, each with a dropdown panel:

1. **Predictions** — Active, expired, and personal predictions, portfolio, margin, create options, and organizations
2. **Exchange** — DEX, instant trade, simple swap, and transfer
3. **Account** — Balances, open orders, and favourites
4. **Settings** — About, nodes, blocked users, create account, and visuals

Each navigation item shows an icon, title, and brief description. The currently active section is highlighted.

## Sidebar

On mobile and smaller screens, the sidebar provides the same navigation in an accordion layout. Toggle it using the hamburger button in the header.

## Language Switcher

Click the globe icon in the header to open the language menu. Select any of the 11 supported languages:

English, Danish, German, Spanish, Estonian, French, Italian, Japanese, Korean, Portuguese, Thai

The UI immediately updates to your selected language. If you are on a documentation page, the language switch redirects you to the localized version of that page.

## Page Layout

Each page follows a consistent structure:

1. **Page header** — Title, description, and navigation
2. **Main content** — The page's primary content area
3. **Page footer** — Source link and additional information
