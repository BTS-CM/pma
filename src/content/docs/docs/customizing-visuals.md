The visual customization page lets you personalize the animated wave background that appears in the page header. Access it from **Settings > Configure Visuals** in the navigation.

## Wave Background

The header features an animated canvas-based wave background. You can customize nearly every aspect of it.

### Wave Settings

- **Wave count** — Number of overlapping waves (more waves = denser animation)
- **Speed** — Animation speed
- **Thickness** — Stroke width of each wave line

### Color Palettes

Choose from several built-in color palettes:

- **Rainbow** — Full spectrum gradient
- **Ocean** — Blues and teals
- **Sunset** — Warm oranges and reds
- **Mono** — Single-color scheme
- **Custom** — Pick your own two colors

### Custom Colors

When using the custom palette, select two hex colors that the waves will interpolate between.

### Aurora

Enable aurora effects for a more dramatic, northern-lights-style animation.

### Particles

Toggle floating particles on or off. Particles add small moving dots that drift across the wave animation.

### Blur

Adjust the blur amount applied to the wave background. Higher values create a softer, more diffused appearance.

## IPFS Gateway

Configure the IPFS gateway URL used to load content such as NFT media and avatars.

- **Default:** `https://ipfs.io/ipfs/`
- Change this if you prefer a different gateway for speed or reliability
- The gateway URL is used when constructing IPFS content links

## Persistence

All visual settings are saved to local storage. Your preferences persist across app restarts and are independent per chain.

## Reset to Defaults

Use the reset button to restore all visual settings to their original defaults.
